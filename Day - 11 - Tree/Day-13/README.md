# 🌳 Day 13 - Binary Search Tree (BST) Deletion


## Topics Covered
- BST Deletion
- Inorder Successor
- Recursive Tree Operations
- Inorder Traversal

## BST Deletion Cases
1. Delete a leaf node.
2. Delete a node with one child.
3. Delete a node with two children using the Inorder Successor.



## Key Learnings
- How BST deletion works for different cases.
- Finding the Inorder Successor (smallest node in the right subtree).
- Maintaining BST properties after deletion.
- Using recursion for tree operations.
