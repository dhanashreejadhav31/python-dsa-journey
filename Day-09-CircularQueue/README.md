# 📚 Day 9 

## 🔹 Topic: Circular Queue Implementation in Python

This project demonstrates the implementation of a **Circular Queue** using arrays (lists) in Python.

---

## 🚀 What is a Circular Queue?

A **Circular Queue** is a linear data structure that follows FIFO (First In First Out), but the last position is connected back to the first position to form a circle.

👉 It overcomes the limitation of a simple queue where space gets wasted after deletions.

---

## 💡 Key Concept

- Uses **modulo operator (%)** for circular movement
- Efficient memory utilization
- Reuses empty spaces
