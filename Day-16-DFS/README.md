# Day 16 - Depth First Search (DFS) in Python


Today, I practiced **Depth First Search (DFS)** using:
- Adjacency Matrix Representation
- Iterative DFS using Stack
- Graph Traversal Techniques

### 💡 Concepts Covered
- Graph Data Structure
- Adjacency Matrix
- DFS Traversal
- Stack Implementation
- Visited Array

### 🚀 Features
- Create an undirected graph
- Add edges between vertices
- Perform DFS traversal from a given source vertex
- Display the order of visited nodes

### 📈 Complexity
- Time Complexity: **O(V²)**
- Space Complexity: **O(V)**
