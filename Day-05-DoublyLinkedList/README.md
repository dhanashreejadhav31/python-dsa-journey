# 📅 Day 5 – Doubly Linked List

## 📌 What is a Doubly Linked List?
A Doubly Linked List is a type of linked list where each node contains:
- Data
- Pointer to the next node
- Pointer to the previous node

This allows traversal in both forward and backward directions.

---

## 🧠 Key Concepts
- Bidirectional traversal
- Efficient deletion (no need to track previous node separately)
- Extra memory usage due to additional pointer

---

## ⚙️ Operations Implemented
- Insertion at beginning
- Insertion at end
- Insertion after a given value
- Deletion of a node
- Traversal

---

## ⚡ Challenges Faced
- Handling `prev` and `next` pointers correctly
- Managing edge cases (head deletion, last node deletion)

---

## 📚 Key Takeaway
Doubly Linked Lists provide more flexibility than singly linked lists but use extra memory.
