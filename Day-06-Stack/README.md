# 📚 Day 6 – Stack Implementation in Python

## 🧠 Topic Covered
Stack Data Structure (LIFO – Last In First Out)

---

## 📖 What I Learned
- Concept of Stack (LIFO principle)
- Basic operations:
  - ✅ Push (Insert element)
  - ✅ Pop (Remove top element)
  - ✅ Peek (View top element)
  - ✅ Length (Size of stack)
- Handling edge cases like **empty stack**

---

## ⚙️ Implementation Approach
Implemented stack using Python list.

### ❌ Initial Approach (Less Efficient)
- `insert(0, value)`
- `pop(0)`
- Time Complexity: **O(n)**

### ✅ Optimized Approach
- `append(value)`
- `pop()`
- Time Complexity: **O(1)**
