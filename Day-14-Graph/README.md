# Day 14 - Graph 

## 📖 What is a Graph?

A **Graph** is a non-linear data structure consisting of:

- **Vertices (Nodes):** The individual elements of the graph.
- **Edges:** The connections between vertices.

Graphs are used to represent relationships between different objects, such as social networks, maps, and computer networks.

---

## 📌 Basic Terminologies

### 1. Vertex (Node)
The fundamental unit of a graph.

### 2. Edge
A connection between two vertices.

### 3. Degree
The number of edges connected to a vertex.

### 4. Path
A sequence of vertices connected by edges.

### 5. Cycle
A path that starts and ends at the same vertex.

### 6. Connected Graph
A graph in which there is a path between every pair of vertices.

---

## 📚 Types of Graphs

### • Directed Graph
Edges have directions.

Example:
A → B

### • Undirected Graph
Edges do not have directions.

Example:
A — B

### • Weighted Graph
Edges have weights or costs associated with them.

### • Unweighted Graph
Edges do not have any weights.

### • Cyclic Graph
Contains at least one cycle.

### • Acyclic Graph
Does not contain any cycle.

### • Connected Graph
Every vertex is reachable from every other vertex.

### • Disconnected Graph
Some vertices are not connected to the rest of the graph.

## 💡 Key Takeaways

- Graphs are non-linear data structures.
- A graph consists of vertices and edges.
- Graphs can be directed or undirected.
- Adjacency Matrix uses O(V²) space.
- Adjacency List is more memory efficient.
- Graphs are widely used in real-world applications.
